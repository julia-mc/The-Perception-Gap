# The-Perception-Gap
### DATA 205 Final Project 
**By Julia Melo Cavalcante**


**Overview**

This project analyzes the gap between what the public reported to 911 calls and what was officially reported by the police after investigation. The goal is to measure the gap between public perception of crime and confirmed criminal activity in Montgomery County and then examine whether that gap is associated with enforcement trends and police districts. 

**Tools Used**

- R Studio
- Power BI
